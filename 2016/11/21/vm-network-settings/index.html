Network settings for virtualbox

2016-11-21
linux, network, vm
  • VM: CentOS 6.5
  • Host: Windows7

VM 設定

  1. 網卡設定

    • 第一張設 NAT
    • 第二張設 橋接介面卡

  2. 確認 VM 網卡設定

    • 確認兩張網卡都有開 eth0 / eth1
    • 沒有的話，就啟動網卡

  3. VM網卡設定

    • eth0 (NAT) 基本上原本的設定就好，不用改

    • eth1 (橋接)

    • 設定完記得重啟

  4. 檢查連線

本機設定

  1. 確定有加網路區段

    區域連線 > IPv4 > 進階 > IP設定

    增加和VM同網段的IP位址 192.168.XXX.LLL

  2. 檢查是否連得到 VM

其他設定

  • 關閉防火牆

  • 設定開機時不會開起防火牆
